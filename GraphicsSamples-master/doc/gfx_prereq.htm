NVIDIA GameWorks Graphics Samples Prerequisites

Windows (Target) Development for OpenGL or Vulkan:

  • Windows Vista, 7, or 8 PC
  • NVIDIA GeForce Fermi-based (GTX 4xx) or better GPU
  • Visual Studio 2013 or 2015, non-Express editions
  • Microsoft DirectX End-User Runtime Web Installer. This is required by all of the OpenGL samples on Windows in order to add XINPUT-based game controller support.
  • LunarG Vulkan SDK. This is required to build and run any of the Vulkan samples.

Android (Target) Development for OpenGL or Vulkan:

  • Windows Vista/7/8/10
  • Tegra 4-based device (advanced GL samples and all Vulkan samples require a Tegra K1/X1 device)
  • Build environment:
    • Windows, Visual Studio:
      • Visual Studio 2013 or 2015, non-Express editions
      • Full Tegra Android Development Pack for Windows tools install
        (Note that NVIDIA Nsight Tegra, Visual Studio Edition should be included in full install if Visual Studio detected)

Linux (Host or Target) Development for OpenGL or Vulkan:

This initial 3.0 release only supports Windows as the development host and Windows or Android as the target. Resumed Linux and Linux4Tegra support is forthcoming.
